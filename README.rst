=======
eekhorn
=======

A fancy SQL console.


Requirements
============

* Python 2.6 or newer (including Python 3)
* ansicolors_
* six
* sqlalchemy_ (0.7 or newer)
* sqlparse_

  
Limitations
===========

eekhoorn currently assumes that your terminal uses UTF-8 as IO encoding.
